# Polymarket Strategy Runs

Per-user signal history, auto-generated by `polymarket-strategy` skill.

## Structure

```
<$USER>/
  <event-slug>/
    <epoch-ms>.ctx.json   — full ctx snapshot at fetch time (markets, orderbooks, klines, news)
    <epoch-ms>.json       — execution log (strategy code + result), written by run_js
```

## Usage

- Every `fetch.ts` invocation creates a `*.ctx.json` in this directory
- Every `run_js` invocation writes the corresponding `*.json` execution log
- Both files share the same timestamp prefix for easy correlation
- Files are git-tracked for cross-user auditability and historical backtesting
